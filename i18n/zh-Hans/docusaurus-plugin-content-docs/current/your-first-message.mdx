# 发送你的第一个跨链消息

本教程演示如何[发送](./reference/messaging/send.mdx)跨链消息到预部署的 `TestRecipient` 合约。

<SimpleMessagingDiagram />

发送消息只需调用 `Mailbox.dispatch`。确保你在源链上有一个带有资金的账户。

<MessageQuickstart />

:::tip
你可以在 [explorer.hyperlane.xyz](https://explorer.hyperlane.xyz/) 上通过搜索你的 `Mailbox.dispatch` 调用的**交易哈希**来查看消息的进度。
:::
