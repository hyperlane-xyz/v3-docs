# Latencies

验证者必须等待一定数量的区块被挖掘，才能被视为有效和[reorg-safe](https://www.alchemy.com/overviews/what-is-a-reorg)。如果没有这个，验证者可能会被减持，因为它们可能已经签署了一个不再有效的检查点。

请参考以下部分，了解Hyperlane验证者使用的区块最终性配置。
