# Send your first interchain message

This tutorial demonstrates how to [send](./reference/messaging/send.mdx) an interchain message to a pre-deployed `TestRecipient` contract.

<SimpleMessagingDiagram />

Sending a message is a simple matter of calling `Mailbox.dispatch`. Make sure you have an account with funds on the origin chain.

<MessageQuickstart />

:::tip
You can watch the progress of your message on [explorer.hyperlane.xyz](https://explorer.hyperlane.xyz/) by searching for the **transaction hash** of your `Mailbox.dispatch` call.
:::
