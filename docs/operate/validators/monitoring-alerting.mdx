# 监控 & 警报

Validators在由参数`--metrics`指定的端口号上公开指标。默认情况下为端口`9090`，但可以自定义任何有效的端口。

:::info

如果你使用Docker镜像来运行，请确保将指标端口进行端口转发。例如，要将宿主机上的端口80转发到Docker容器内部的端口9090，你可以在docker run命令中添加以下标志：-p 9090:80。

:::

我们还提供了一个准备就绪的Grafana仪表板，可帮助您快速入门，您可以在[tools/grafana](https://github.com/hyperlane-xyz/hyperlane-monorepo/tree/main/tools/grafana)下找到源代码和导入说明。如果您想使用自己的仪表板，则`hyperlane_latest_checkpoint`是两个最关键的度量值，分别为`phase="validator_observed"`和`phase="validator_processed"`。它应该会逐渐增加，而且这两个值不应该出现不同步的情况。
