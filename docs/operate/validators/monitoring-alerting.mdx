# 监控 & 警报

Validators在由参数`--metrics`指定的端口号上公开指标。默认情况下为端口`9090`，但可以自定义任何有效的端口。

:::info

如果你使用Docker镜像来运行，请确保将指标端口进行端口转发。例如，要将宿主机上的端口80转发到Docker容器内部的端口9090，你可以在docker run命令中添加以下标志：-p 9090:80。

:::

## Metrics
 
- “`hyperlane_latest_checkpoint` 是 `phase="validator_observed"` 和 `phase="validator_processed"` 这两个维度中最关键的指标。它应该逐渐增加，并且这两者不应该出现明显的不同步。”
- 另一个有用的指标是 `hyperlane_block_height`，它表示验证者连接到的 RPC 节点的区块高度。
